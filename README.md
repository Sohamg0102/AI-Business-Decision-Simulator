
# 📊 AI Business Decision Simulator

An AI-powered web application that helps businesses simulate decisions, analyze outcomes, and identify optimal strategies using machine learning.

---

## 🚀 Features

- 🔐 User Authentication (Login / Signup)
- 🧠 Machine Learning Revenue Prediction Model
- 📊 Scenario Simulation (Price, Marketing, Customers)
- 📈 Interactive Insights Dashboard
- ⚡ Auto Simulation Engine (Multiple Scenarios)
- 🎯 Strategy Recommendation Engine (Optimization)

---

## 🧠 How It Works

1. User inputs business parameters:
   - Price
   - Marketing Spend
   - Customers

2. ML model predicts revenue

3. System:
   - Compares strategies
   - Stores results in database
   - Generates insights

4. AI engine suggests:
   - Best price
   - Best marketing spend
   - Optimal customer target

---

## 🏗️ Tech Stack

- **Frontend:** Streamlit  
- **Backend:** Python  
- **Database:** SQLite  
- **ML Model:** Scikit-learn  
- **Visualization:** Plotly
